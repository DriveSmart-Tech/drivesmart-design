DriveSmart

Design System

Dark glassmorphism · DriveSmart-blue accent · zero build, zero framework. One drivesmart.css carries the tokens, utilities, and every component below. Press ⌘K to jump anywhere.

vanilla CSSno Reactno Tailwindno buildlight + dark

Get started

Installation

Three ways in — drop the CSS into any project, install the Claude Code skill, or hand the brand guide to a designer. No build step, no dependencies.

1 · Drop-in CSS any project

Copy drivesmart.css into your project and link it — that's the whole system. Fonts are @imported inside it. Or hit CSS (top-right of the topbar) to copy the entire file.

<link rel="stylesheet" href="drivesmart.css">
<!-- icons (optional) -->
<link href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined" rel="stylesheet">

<body class="bg-grid-pattern">
  <div class="aurora-bg"></div>
  <div class="glass-card card-mesh-success p-5">
    <div class="metric text-success">12,847</div>
    <div class="label-caps text-2 mt-2">Paid</div>
  </div>
</body>

Want a full-page skeleton (sidebar + topbar + content) to start from? Copy starter.html.

2 · Claude Code skill marketplace

The system also ships as a Claude Code plugin — the /drivesmart-design skill — distributed through the DriveSmart marketplace (DriveSmart-Tech/drivesmart-marketplace, the home for all DriveSmart plugins / skills / agents). In any Claude Code session:

/plugin marketplace add DriveSmart-Tech/drivesmart-marketplace
/plugin install drivesmart-design@drivesmart

Then the skill drives the system for you:

CommandWhat it does
/drivesmart-designDrop the system into the current project + scaffold an app-shell.
/drivesmart-design auditRead-only audit of an existing UI vs the system → DRIVESMART-REVIEW.md.
/drivesmart-design migrateAudit, then migrate the UI onto the system surface-by-surface.
/drivesmart-design referenceAnswer “what class / token do I use for X”.

3 · Designers & marketing no code

Not a developer? The Brand guide is the whole look in plain language — every colour (with copy-paste hex), font and size for Figma, Canva, slides, social or print. The full technical Spec renders the DESIGN.md reference.
