Design System

Dark glassmorphism · DriveSmart-blue accent · zero build, zero framework. One drivesmart.css carries the tokens, utilities, and every component below. Toggle theme & roundness above — the whole UI re-derives from a handful of base hexes. Hit Show code on any demo to copy the markup.
